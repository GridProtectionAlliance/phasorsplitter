To install the 64-bit version of Stream Splitter, extract and run
"StreamSplitterSetup (64-bit).msi". To install the 32-bit version of
Stream Splitter, extract and run "StreamSplitterSetup (32-bit).msi".
The MSI may also be used to upgrade an existing version of Stream Splitter.
Configuration will be preserved across installations.
